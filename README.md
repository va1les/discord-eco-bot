# install
### 📨 **File** `example.env` rename to `.env`
```Shell
DISCORD_TOKEN=YourTokenHere
MONGO=YourMongoUriHere
```

### 📥 Install all modules — `npm install`

### ✅ Launch the Bot — `node .`
